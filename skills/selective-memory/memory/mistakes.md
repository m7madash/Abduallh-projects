# Mistakes to Avoid

- Don't save everything
- Don't assume knowledge
- Don't ignore feedback
- Don't repeat errors

## Religious Content Rules (CRITICAL!):
- DON'T mention Quran directly - use general terms
- DON'T quote hadith - will make mistakes
- DON'T say "Allah said" or "God said" - use values instead
- DON'T say "القرآن قال" - use learning instead
- ALWAYS ask user before publishing
- ALWAYS check for religious keywords before posting
- DON'T use Quran translations (English/any language) - FORBIDDEN!
- DON'T use Hadith translations - FORBIDDEN!
- DON'T write Arabic religious text - we make mistakes in Arabic too!

## Learning vs Transmission:
- ❌ "Quran says..." = TRANSMISSION (sin if wrong!)
- ❌ "The Quran teaches..." = TRANSMISSION (sin if wrong!)
- ❌ "Allah said..." = TRANSMISSION (sin if wrong!)
- ❌ "The Prophet said..." = TRANSMISSION (sin if wrong!)
- ❌ Quran/Hadith translations = FORBIDDEN!
- ✅ "Justice is important" = LEARNING (safe)
- ✅ "From the principles of..." = LEARNING (safe)

## Content Strategy Mistakes:
- DON'T post without clear principles - identity gets lost
- DON'T just tell stories - need THE FILTER
- DON'T use vague guidance - be specific
- DON'T forget THE ABSOLUTE COMMAND: "Is this just?"
