# 🛡️ مشروع "وضع غض البصر" — Modesty Filter for Social Platforms

## 🎯 الهدف
حماية المستخدمين (خاصة الرجال) من المحتوى النسائي المثير عبر فلترة ذكية للمحتوى على منصات التواصل، تمكيناً لأمر الله: «غضوا من أبصاركم».

## 📦 المكونات

- **`core/modesty_filter.py`** — خوارزمية الفلترة الرئيسية (Python)
- **`config/settings.json`** — إعدادات المستخدم (تفعيل/تعطيل، الجنس، الوضع الصارم)
- **`models/`** — (قيد التطوير) نماذج تصنيف الصور/النصوص
- **`missions/modesty_mode_ar.md`** — منشور توعوي للنشر على المنصات

## 🚀 الاستخدام الأساسي

### **CLI:**
```bash
# تفعيل الوضع
python -m modesty_filter --enable --gender male

# الوضع الصارم (حظر إضافي)
python -m modesty_filter --strict

# عرض الإعدادات
python -m modesty_filter --status

# اختبار منشور
python -m modesty_filter --test-post sample_post.json
```

### **كود Python:**
```python
from modesty_filter import ModestyFilter

filter = ModestyFilter()
filter.config.user_gender = "male"
filter.config.enabled = True

post = {
    "category": "fashion",
    "tags": ["women", "style"],
    "text": "موديلات صيفية جديدة"
}
if filter.should_hide_post(post):
    print("يتم إخفاء هذا المنشور")
```

## 🔐 الخصوصية

- **لا تخزين بيانات شخصية** — الإعدادات محلية.
- **التصنيف محلي** إن أمكن (on-device) لتقليل إرسال الصور.
- **شفافية:** المستخدم يتحكم في الإعدادات ويرى ما تم حظره.

## 🕌 الضوابط الشرعية

✅ **موافق للشرع:**
- يساعد على غض البصر (أمر قرآني مباشر: النور:30).
- يقلل الإثارة ويحمي من الفتنة.
- لا يحظر المحتوى التعليمي (صحة، دين، وعي).

❌ **يجب تجنبه:**
- فرض الحلال على غير المسلمين (ال 특务 اختياري).
- منع محتوى نسائي تعليمي (صحة المرأة، الأسرة مسموح).
- استخدام النظام للرقابة السياسية أو التعصب.
