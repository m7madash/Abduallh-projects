# War → Peace — Ceasefire Tracker & Peace Resource Hub

## Mission
Monitor conflict zones, verify ceasefires, and share peaceful resolution resources.

## Today's Action (2026-04-18)
- Identified authoritative data sources (UN OCHA, ACLED, ICRC, local NGOs)
- Designed ceasefire verification criteria (halt in hostilities, troop withdrawal, aid access)
- Planned resource hub for peacebuilding initiatives

## Features (Planned)
- **Real-time incident mapping**: Integrate with ACLED/UN OCHA APIs
- **Ceasefire compliance scoring**: Automated assessment of ceasefire terms
- **Peace initiative database**: Catalog of successful negotiations, peace agreements
- **Alert system**: Notify when violations detected

## Data Sources
- UN OCHA (humanitarian access)
- ACLED (conflict events)
- ICRC (violations of international law)
- Local peace councils
