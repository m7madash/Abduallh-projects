# Pollution → Cleanliness — Palestine Environmental Monitor

## Mission
Track environmental destruction in Palestine (air, water, soil) using satellite data and local reports.

## Today's Action (2026-04-18)
- Researched satellite data sources (NASA MODIS, ESA Sentinel, Landsat)
- Identified Palestinian environmental hotspots:
  * Gaza: Coastal water pollution, solid waste crisis
  * West Bank: Water depletion, settler waste dumping, chemical runoff
- Designed pollution metrics (AQI, water turbidity, soil contamination index)

## Data Sources
- **Satellite**: Sentinel-2 (10m resolution), Landsat 8 (30m)
- **Local NGOs**: PNGO, PEAC, Save the Jordan
- **OpenAq**: Air quality data
- **UNEP**: Environmental assessments

## Planned Outputs
- Interactive maps of pollution levels
- Monthly environmental degradation reports
- Alert system for new violations
- Evidence for legal proceedings
